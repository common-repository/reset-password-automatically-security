Resets password of administrator accounts automatically on pre-defined time set by admin in settings and mail new password to registered email of user.

== Description ==

Plugin is helpful if you wish to keep you website secure from hack attack and password theft. 

<strong>Features:</strong>
<ul>
<li>Change password of admin account automatically on defined time intervals by admin user. New password is send to registered email of administrator.</li>
<li>In case of more than one admin accounts, password of all admin accounts are changed and new password is mailed to individual admin's registered email address</li>
<li>Password change frequency can be set as any day per your requirement</li>
<li>Capable to generate strong password with combination of alphabets, numbers and special characters</li>
<li>Option to generate strong yet pronounceable passwords</li>
</ul>

== Installation ==

1. Unzip plugin in `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. open page from Settings -> Password Generator
